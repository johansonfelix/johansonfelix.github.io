<!-- GETTING STARTED -->
## Getting Started

To setup the application on your local environment: 


### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/johansonfelix/johansonfelix.github.io.git
   ```
2. Install dependencies
     ```sh
    yarn install
   ```

4. Start react
 ```sh
    yarn start
   ```
5. Test application on localhost:3000
